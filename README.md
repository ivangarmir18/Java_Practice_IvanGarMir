# Java_Practice_IvanGarMir

*[Read in English below](#english-version)*

Este repositorio es mi espacio de trabajo personal dedicado al aprendizaje progresivo de Java. 

El objetivo del repositorio es documentar mi evolución práctica en el lenguaje partiendo completamente desde cero. A través de retos y ejercicios diarios, el nivel de complejidad irá subiendo gradualmente: desde la sintaxis más básica y el tipado estricto, hasta dominar la Programación Orientada a Objetos (POO), la gestión de memoria y el control de hardware simulado. 

Es un registro vivo de mi transición desde lenguajes de tipado dinámico (Python) hacia arquitecturas de software más robustas y estructuradas.

This repository is my personal workspace dedicated to the progressive learning of Java.

The goal of this repository is to document my practical evolution in the language, starting entirely from scratch.
